# Python CLI Calculator

This repository contains a simple command-line calculator written in Python.
It shows my learning progress by including an initial version and an improved version.

## Versions

### Version 1 – Basic Calculator
File: `Calculator_v1_basic.py`

- Simple arithmetic operations
- Minimal input handling
- Linear logic structure
- Created as my first working calculator

### Version 2 – Improved Calculator
File: `calculator_v2_improved.py`

Improvements made:
- Modular code using functions
- Input validation with error handling
- Additional operations (power, modulo)
- Menu-driven interface
- Cleaner and more readable structure

## Purpose
This project was created as a beginner exercise to practice:
- Python fundamentals
- Functions and control flow
- Writing cleaner, more maintainable code
